# Пользователи

Пользовательские операции выполняются в Web App админке. Доступ получают только Telegram-пользователи из `ADMIN_IDS`.

## Что доступно администратору

- список пользователей с поиском и фильтрами;
- просмотр подписки, статуса, трафика и premium-лимитов;
- блокировка пользователя;
- ручная синхронизация с Remnawave Panel;
- тикеты поддержки и ответы пользователю;
- просмотр платежей и служебных событий.
